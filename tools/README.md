# Tools folder

Students place the prebuilt GTA Synthea JAR in this folder.

Large JAR files, cloned upstream repositories, downloaded postal-code data,
and build artifacts are intentionally excluded from normal Git history.

The official student JAR will be distributed through the GitHub Releases page.

1. Download the GTA Synthea JAR from the repository's Releases page.
2. Place it directly inside this tools folder.
3. Do not rename it unless the project instructions say otherwise.
